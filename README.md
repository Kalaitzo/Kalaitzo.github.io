# kalaitzo.github.io

My personal portfolio website — an interactive 3D side-scroller where a bipedal robot walks along a path and discovers content at themed stations.

**[kalaitzo.github.io](https://kalaitzo.github.io)**

## About

I'm Vasileios Kalaitzopoulos, an Electrical & Computer Engineer (MEng, University of Patras) with a focus on robotics, computer vision, and software engineering. The robot in this portfolio is based on the real 6-DOF biped I built for my thesis on bipedal locomotion via reinforcement learning.

## Navigation

- **Arrow keys** or **A/D** to walk
- **Enter** at a station to view content
- **Hamburger menu** for full CV
- Touch controls on mobile

## Built With

- [Three.js](https://threejs.org/) — 3D rendering
- [Vite](https://vite.dev/) — build tool
- Vanilla JavaScript
- Deployed on GitHub Pages

## Local Development

```bash
npm install
npm run dev
```
